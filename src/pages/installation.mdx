## Basic usage

1. Define your schema:

```json copy
{
    "fields": [
        {
            "name": "first_name",
            "meta": {
                "displayType": "text",
                "displayName": "First Name"
            }
        },
        {
            "name": "last_name",
            "meta": {
                "displayType": "text",
                "displayName": "Last Name"
            }
        }
    ]
}
```

2. Import and use **MuiForms**:

```tsx copy
import MuiForms from "mui-forms";
import schema from "<schema_location.json>";

<MuiForms
    name="myForm"
    schema={schema}
    onChange={(...data) => {
        // handle data on change
    }}
    onError={(...fieldData) => {
        // handle error
    }}
    onNext={(formData) => {
        // handle next
    }}
    onSubmit={(formData) => {
        // handle form submission
    }}
/>;
```
