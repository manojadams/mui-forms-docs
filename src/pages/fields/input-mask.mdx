# Input-mask field

### Display type

- input-mask

### Config
|Name |Description|Notes|
|-----|-----------|-----|
|mask|Mask string.| [Details here](https://www.npmjs.com/package/react-input-mask#mask--string)|
|maskChar|Character to cover unfilled parts of the mask. [Details here](https://www.npmjs.com/package/react-input-mask#maskchar--string)|Default character is "_"|
|formatChars|Defines format characters with characters as a keys and corresponding RegExp strings as a values.| [Details here](https://www.npmjs.com/package/react-input-mask#formatchars--object)|
