# File field

### Display type

- file

### Config

|Name |Description|Notes|
|-----|-----------|-----|
| accept | Type of file to be accepted|For example: image/jpeg,image/png|
| filePreview | Whether to preview file in display. Enabled for images only|Type: boolean|
| blob | Whether to submit data in binary format on form submission. Else it will be converted to **base64** format|Type: boolean|
| multiple | Whether to allow multiple file uploads |Type: boolean |
