## Basic properties

| Name               | Description                                                                                                                                                                                                                                              | Type      |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| className          | CSS classname |
| name               | The name of the form                                                                                                                                                                                                                                     | Optional  |
| config             | Configuration options for the form. [Read more](/config)                                                                                                                                                                                                                  | Optional  |
| data               | Contains initial form data values with which form is loaded                                                                                                                                                                                              | Optional |
| schema             | JSON based schema to render the form. [Read more](/schema)                                                                                                                                                                                                                     | Mandatory |
| controls           | Custom fields. [Read more](/custom-fields/about)                                                                                                                                                                                                                                            | Optional  |
| components         | Custom template components mapped with schema using displayType property as template. [Read more](/custom-fields/about#components)                                                                                                                                                                                                                               | Optional  |
| pageNumber         | The default tab number to be open for a grouped form |Optional|
| lastPageNumber     | Last page number will ignore any tabs appearing after the lastPageNumber tab | Optional |
| onChange           | handle to be called on field value change. Second parameter contains all of the form data | Optional |                                                                                                                                                                                                                | Optional  |
| onError            | handle to be called on field error                                                                                                                                                                                                                       | Optional  |
| onNext             | handle to be called when user tries to move to next form section (grouped forms). For async data, use property `useNextResponse`                                                                                                                                                                          | Optional  |
| onPrevious         | handle to be called when user tries to move to previous form section (grouped forms)                                                                                                                                                                     | Optional  |
| onSubmit           | handle to be called on form submission                                                                                                                                                                                                                   | Mandatory |
| changeResponseMode | Value: "form-data", "section-data", "default". <br /> **`form-data`**: onChange event will pass all the form data as a second parameter. <br />**`section-data`**: onChange event will pass all the section data as a second parameter.                  | Optional  |
| nextResponseMode   | Value: "form-data", "page-data" <br/> **`form-data`**: onNext event will send all the form data (onNext). <br /> **`page-data`**: onNext event will send all the date of the current page. <br/> `Note`: onNext is triggered on click of **Next** button | Optional  |
| useNextResponse    | Setting to `true` makes the onNext call to wait for a response from server | Optional |
