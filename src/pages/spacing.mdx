# Field Gaps

Config properties like `gapX` and `gapY` can be used to customize field gaps/spacing.

import Spacing from "@/components/Spacing";

## Default

<Spacing gapX={1} gapY={1} />

## More gaps

<Spacing gapX={2} gapY={2} />

### Usage

```tsx
    <MuiForms
        config={{
            gapX: 2,
            gapY: 2
        }}
        schema={schema}
        onSubmit={formData => {
            // TO DO
        }}
    />
```
